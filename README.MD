# LeadScore Engine (Chrome Extension) (Beta Version)

A smart Chrome extension (beta version) that extracts high-quality leads from Google Maps and enriches them with contact details and social links.

## Features

* Extract top-rated leads from Google Maps
* Smart ranking based on rating + reviews
* Enrich leads with:

  * Website
  * Email
  * Phone
  * Social media links (Facebook, Instagram, LinkedIn)
* CSV export (based on plan)
* License-based usage control

Advanced features and higher limits available in premium plans

## Installation (Manual)

1. Download or clone this repository
2. Open Chrome and go to:
   chrome://extensions/
3. Enable "Developer mode"
4. Click "Load unpacked"
5. Select the downloaded folder

## Usage

1. Enter a search like:

   * "dentists in New York"
   * "real estate agents near me"
2. Click "Find Leads"
3. Click "View Details" for enriched contact data

## Notes

* Requires a valid license key for full functionality
* Free version available with limited usage
